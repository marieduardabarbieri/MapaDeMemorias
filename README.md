# MapaDeMemorias
O aplicativo "Mapa de Memórias" foi desenvolvido com o objetivo de centralizar essas recordações em um único lugar, permitindo que o usuário registre suas experiências de viagem de forma organizada e visual.
![Uploading image.png…]()

# O sistema possibilita cadastrar:

País visitado;

Cidade;

Data da viagem;

Descrição da experiência;

Fotografia do momento registrado.

Além disso, o aplicativo apresenta uma interface intuitiva e organizada para consulta dos registros realizados.

# Tecnologias Utilizadas

React Native

Expo

TypeScript

React Navigation

SQLite

Expo Image Picker (para selecionar fotos)

Hooks (useState e useEffect)

| Requisito       | Como foi atendido                  |
| --------------- | ---------------------------------- |
| Componentização | Componentes reutilizáveis          |
| Navegação       | Tela Inicial → Cadastro → Memórias |
| Hooks           | useState e useEffect               |
| SQLite          | Armazenamento local                |
| Hardware        | Câmera/Galeria                     |
| GitHub          | Repositório público                |
